Unit: 2
Theme: Exploratory Data Analysis

Weekly focus

This unit focused on exploratory data analysis and the importance of understanding a dataset before modelling. The main lesson was that machine learning quality depends heavily on data quality, feature understanding and the identification of anomalies.

Commentary on learning

What changed for me in this unit was recognising that EDA is not just a preliminary step but a core analytical process. It shapes which variables are trusted, how outliers are treated and whether the final model can be interpreted responsibly.
